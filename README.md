# ⏰ Digital Clock

## 📌 Description
This project is a real-time digital clock built using web technologies. It dynamically displays the current time, including hours, minutes, seconds, and AM/PM format. The application updates every second, demonstrating time-based functions and DOM manipulation in JavaScript.

## 🚀 Features
- Real-time clock updates every second
- Displays hours, minutes, and seconds
- 12-hour format with AM/PM indicator
- Clean and simple user interface

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## ▶️ How to Run
1. Clone the repository  
2. Open the project folder  
3. Run `index.html` in your browser
